# forma11y known as VisionImpairedFriendlyMedicalForms
Decrease barriers for blind and vision impaired people in medical practices. 

## Requirments
- Node
- Ionic (Angular)

## Get started
- clone
- npm ci
- npx ionic serve


## Must have
- Web forms
- Form Validation
- 0 Errors on HTML Codesniffer / pa11y plugin / Netlify a11y plugin
